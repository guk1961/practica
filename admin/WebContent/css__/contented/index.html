Introduction to Contented6

Contented6 is the sixth open source template from Contented Designs.

A slogan or contact information can be displayed inside the horizontal blue line just below the header by using the slogan tag as shown in the comments in index.html. The area will expand its height to fit the text.

The Contented6 template is valid CSS, valid XHTML 1.0 strict, and it's been tested for consistent appearance on:

  • Internet Explorer 6.0.2900.2180
  • Firefox 1.5.0.3
  • Opera 8.53

Current version: Contented6 Version 1.01 (May 17, 2006)

Using this Design

Contented6 is an open source template. You're free to modify it and use it for any purpose without cost or obligation. We prefer that you leave the link to our website in the footer but it's not required.

If you have comments or questions, please contact us at Contented Designs. Thanks!
